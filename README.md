# Console_Game_Hangmen
ПРИВЕТ ВСЕМ!
это мой первый проект который я написал от идеи до реализации полностью сам.
игра виселица не нова, зная игровые правила я решил написать собственную реализацию этой игры
в игре есть 100 слов которые нужно угадать
каждое слово это отдельный уровень
также за правильно отгадонную букву начисляются баллы, за пройгрыш отнимаются баллы.
игра написана на с++, без использования ООП, чистый спагети код
